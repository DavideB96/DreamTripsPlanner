# Dream Trips Planner ✈️

A modern travel planning web app built with React.

Users can explore destinations, filter by continent and budget, view detailed itineraries, and save their favorite trips.

Live demo: https://dream-trips-planner.vercel.app

## 🚀 Features

- Search destinations
- Filter by continent and budget
- Favorite system with local storage
- Detailed pages with itineraries
- Contact section for travel assistance
- Responsive design

## 🛠 Tech Stack

- React
- JavaScript (ES6+)
- CSS3
- React Router
